# Instalacja Oracle Linux 7.9 oraz Oracle Database 12c R1 12.1.0.2

ERP Egeria 6 oficjalnie działa najwyżej na Oracle Database 12c R1 12.1.0.2, a ta działa najwyżej na Oracle Linux 7.9.

Można zainstalować na Oracle Linux 8, ale wymaga to dodatkowych czynności.  
[Oracle® Database Release Notes 12c Release 1 (12.1) for Linux](https://docs.oracle.com/en/database/oracle/oracle-database/12.1/ladbn/toc.htm#BABDCHGH) punkt 2.6
