# ST - Środki Trwałe
- **ST_numery_seryjne.sql** - Numery seryjne środków trwałych dłuższe niż 18 znaków
- **ST_zalaczniki.sql** - Lista załączników powiązanych ze środkami trwałymi
- **ST_zalaczniki_blob.sql** - Unikalne załączniki BLOB dla środków trwałych
- **ST_zalaczniki_blob_cx_oracle.py** - Pobiera załączniki z bazy danych Oracle i zapisuje je na dysku (moduł cx_Oracle)
- **ST_zalaczniki_blob_oracledb.py** - Pobiera załączniki z bazy danych Oracle i zapisuje je na dysku (moduł oracledb)

# ST - Fixed Assets
- **ST_numery_seryjne.sql** - Serial numbers of fixed assets longer than 18 characters
- **ST_zalaczniki.sql** - List of attachments related to fixed assets
- **ST_zalaczniki_blob.sql** - Unique BLOB attachments for fixed assets
- **ST_zalaczniki_blob_cx_oracle.py** - Downloads attachments from the Oracle database and saves them to disk (cx_Oracle Python module)
- **ST_zalaczniki_blob_oracledb.py** - Downloads attachments from the Oracle database and saves them to disk (oracledb Python module)
